# Heart-Failure-Prediction
This project demonstrates data analysis and preprocessing using Python in a Jupyter Notebook environment. It focuses on loading datasets, cleaning data, performing exploratory data analysis (EDA), and generating meaningful insights using visualizations and statistical techniques.

Objectives Load and explore datasets using Python Clean and preprocess raw data Perform exploratory data analysis (EDA) Apply basic statistical analysis Visualize data trends and patterns Export cleaned data for future use

Technologies: Machine Learning, Python, Pandas, NumPy, Scikit-learn, Matplotlib, Jupyter Notebook
